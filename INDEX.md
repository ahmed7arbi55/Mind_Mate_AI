## 🎯 كيفية استخدام هذه الملفات

### للبدء السريع:
1. اقرأ `README.md` للنظرة العامة
2. شغّل `run.py` لبدء الخادم
3. زر `http://127.0.0.1:8000/docs` لاختبار الـ API

### للتطوير:
1. اقرأ `app/main.py` لفهم البنية
2. عدّل الكود حسب احتياجاتك
3. استخدم `test_api.py` للاختبار
4. راجع `MIGRATION_GUIDE.md` لفهم البنية

### للإنتاج:
1. استخدم `.env` لتعيين الإعدادات
2. استخدم `Dockerfile` و `docker-compose.yml`
3. راجع `API_DOCS.md` لتوثيق الـ API

### للتعلم:
1. ابدأ بـ `README.md`
2. اقرأ `MIGRATION_GUIDE.md` لفهم التحويل
3. اطلع على `EXAMPLES.py` لأمثلة عملية
4. راجع `WEBSOCKET_EXAMPLE.md` للميزات المتقدمة
